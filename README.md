# EmpCo

Eine Sammlung von Open-Source-Tools zur Überprüfung touristischer Inhalte nach EmpCo-Richtlinien.

## Ziel des Repositories

Dieses Repository soll den Austausch von Code, Ideen, Prüfmechanismen und praktischen Ansätzen rund um EmpCo ermöglichen.

Alle Beiträge dienen als Diskussions- und Arbeitsgrundlage. Sie können genutzt, angepasst, erweitert oder kommentiert werden.

## Nutzung

Die bereitgestellten Inhalte können frei eingesehen, getestet und weiterentwickelt werden. Vor einer produktiven Nutzung sollte jeder Code eigenständig geprüft werden.

## Austausch und Beiträge

Beiträge sind willkommen, zum Beispiel durch:

- neue Tools oder Skripte
- Verbesserungen bestehender Ansätze
- Hinweise auf Fehler oder Unklarheiten
- Vorschläge zur Weiterentwicklung
- Issues oder Pull Requests

Bitte beschreibe bei Beiträgen möglichst klar, welchen Zweck der Code erfüllt und welche Annahmen oder Grenzen bestehen.
